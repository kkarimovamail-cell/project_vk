# Предсказание склонности к секретному действию
*Predicting probability of secret event*

## О проекте

Данный проект представляет собой решение задачи бинарной классификации для прогнозирования склонности пользователя к совершению целевого действия. Проект выполнен в рамках прикладного проекта для НИУ ВШЭ. В качестве входных данных взяты данные с VK Edication и хакатона VK Predict CMC Hackathon на платформе Kaggle 

**Основные характеристики:**
- **Цель:** предсказать вероятность совершения пользователем целевого действия
- **Объем выборки:** 247 972 строки, 1 360 признаков
- **Дисбаланс классов:** целевое событие составляет всего 1,35%
- **Результат:** лучшая модель LightGBM достигла ROC-AUC = 0,6868

## Описание данных

**Источник:** хакатон VK Predict CMC Hackathon (платформа Kaggle)

**Характеристики выборки:**

| Параметр | Значение |
|----------|----------|
| Объем обучающей выборки | 247 972 записи |
| Объем тестовой выборки | 106 274 записи |
| Исходные признаки | 1 360 (все анонимизированы) |
| Доля целевого события | 1,35% (3 346 записей) |

Данные представляют собой анонимизированные числовые признаки. Значение `-1` используется для кодирования отсутствия информации.

## Методология

### 1. Feature Engineering (создание признаков)

Выполнено **7 типов преобразований**:

| Преобразование | Описание |
|----------------|----------|
| **Бинаризация -1** | Флаги для двух признаков, где -1 даёт сдвиг вероятности целевого действия на 15–17% |
| **Бинаризация ненулевых** | Флаги для 30 признаков с сильным эффектом ненулевого значения |
| **Логарифмирование** | `log1p(x + shift)` для 50 скошенных признаков |
| **Frequency encoding** | Замена значений на частоту встречаемости |
| **Попарные взаимодействия** | 70 новых признаков из произведений топ-20 признаков |
| **PCA** | 20 компонент, объясняют 43,23% дисперсии |
| **Кластеризация** | MiniBatchKMeans (30 кластеров) + расстояние до центра |

**Итог:** количество признаков выросло с 1 360 до 1 584.

---

### 2. Модели

Были обучены и сравнены три модели:

| Модель | ROC-AUC |
|--------|---------|
| **LightGBM** | **0,6868** |
| Logistic Regression | 0,6370 |
| CatBoost | 0,5901 |

Для оценки использовалась 5-кратная стратифицированная кросс-валидация с усреднением по двум seed-значениям.

---

### 3. Гипотезы (подтверждены)

- Бинаризация -1 повышает качество модели
- Бинаризация ненулевых значений повышает качество
- Логарифмирование скошенных признаков улучшает качество
- PCA и кластеризация не ухудшают качество

---

### 4. Сложности

| Проблема | Описание |
|----------|----------|
| **SMOTE** | Переполнение памяти — генерация миллионов синтетических примеров |
| **AutoGluon** | Требовал >13 ГБ RAM |
| **CatBoost** | Результат 0,5901 — алгоритм заточен под категориальные признаки |

---

## Результаты

- **Лучшая модель:** LightGBM solo
- **ROC-AUC:** 0,6868
- **Сравнение с baseline:** превосходит Logistic Regression (0,6370) на **+7,8%**
